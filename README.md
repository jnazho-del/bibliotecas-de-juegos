# 🎮 Biblioteca Gamer SPA - Sumativa 3

Proyecto de Single Page Application (SPA) desarrollado con **React**, **Vite** y **Bootstrap 5**. Esta aplicación permite explorar una biblioteca de videojuegos gratuita, gestionar favoritos y redactar reseñas personales con persistencia local.

## 🚀 Características
- **Consumo de API**: Datos en tiempo real desde Free-To-Game API.
- **CRUD Completo**: Añadir, leer, editar y eliminar reseñas de juegos favoritos.
- **Persistencia**: Uso de `LocalStorage` para mantener tus datos tras recargar la página.
- **Diseño Premium**: Paleta de colores Naranja/Amarillo, responsive y con efectos modernos.
- **Búsqueda Avanzada**: Filtrado por nombre y género en tiempo real.

## 🛠️ Tecnologías
- **Core**: React 18 + Vite
- **Estilos**: Bootstrap 5 + Custom CSS
- **Iconos**: Lucide React + Bootstrap Icons
- **Peticiones**: Axios
- **Persistencia**: Local Storage API

## 📦 Instalación

1.  Asegúrate de tener **Node.js** instalado.
2.  Navega a la carpeta del proyecto:
    ```bash
    cd biblioteca-videojuegos
    ```
3.  Instala las dependencias:
    ```bash
    npm install
    ```
4.  Inicia el servidor de desarrollo:
    ```bash
    npm run dev
    ```
5.  Abre `http://localhost:5173` en tu navegador.
